# AstroQuant-BTC
AstroQuant-BTC: Pipeline de Big Data &amp; IA. Utiliza DuckDB e SQL avançado (Window Functions) para processar indicadores técnicos em alta performance. O "core" do projeto é uma rede neural LSTM (TensorFlow) multimodal, que integra preço e RSI para prever tendências do Bitcoin, validada por um sistema de backtesting de estratégias quantitativas.
